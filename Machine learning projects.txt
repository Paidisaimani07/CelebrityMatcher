## ⭐ The most important rule

For **every algorithm**, follow this exact process:

```text
                ALGORITHM
                    │
        ┌───────────┴───────────┐
        │                       │
   Understand                 Code
        │                       │
 Mathematics              From Scratch
        │                       │
        └───────────┬───────────┘
                    ↓
             Scikit-learn
                    ↓
             Real Dataset
                    ↓
             Real Project
                    ↓
           Compare Algorithms
                    ↓
              Tune Model
                    ↓
                Deploy
```

**Don't just run `model.fit()` and move on.** For the important algorithms, first implement a simplified version **from scratch**, then use the professional library implementation. That's where you'll truly understand what's happening internally.
